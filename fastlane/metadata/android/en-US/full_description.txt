FOSS Warn is an app for receiving global emergency alerts. Whether it's flooding, a tsunami, or bomb disposal, if there's an alert, you'll receive it.

FOSS Warn uses the FOSS Public Alert Server to obtain alerts. This aggregates official feeds with alerts from over 100 countries worldwide.

A list of supported countries can be found here: https://alerts.kde.org/sources/status

Features:
- Receive alerts in over 100 countries worldwide
- Receive push notifications via UnifiedPush
- Display alerts on a map
- Material 3

FOSS Warn and the FOSS Public Alert Server are independent and in no way affiliated with any official authority.
